Confluence JSON Editor

A Chrome extension that provides direct JSON editing capabilities for Confluence pages via the REST API, enabling advanced page modifications and bulk find/replace operations.

Overview

This Chrome extension opens a side panel that allows you to view and edit the raw JSON data of Confluence pages. It's particularly useful for making precise edits, bulk find/replace operations, or working with page structures that are difficult to modify through the standard Confluence editor.

    Technologies

  • Platform: Chrome Extension
  • Architecture: Side panel with background service worker
  • API: Confluence REST API v1

    • Permissions

  • activeTab: Access to current Confluence page
  • scripting: Page interaction capabilities
  • sidePanel: Side panel UI display
  • storage: Secure credential storage
  • host_permissions: Limited to configured Confluence domain

    • Architecture

  • Background Worker: Handles extension lifecycle
  • Side Panel: Main editor interface
  • Settings Page: Credential configuration UI
  • Storage: Chrome sync for cross-device settings

    • Use Cases

    Content Migration

  • Export page JSON for backup or migration
  • Modify page structure before import
  • Bulk update page properties

    • Bulk Edits

  • Find and replace text across entire page JSON
  • Update multiple instances of links or references
  • Modify macro parameters in bulk

    • Debug & Troubleshooting

  • Inspect page structure and hidden metadata
  • Diagnose rendering issues
  • View raw storage format and version information

    • Advanced Modifications

  • Edit page properties not accessible via UI
  • Modify macro configurations directly
  • Update metadata and custom properties

    • Configuration Required

    To use this extension, you need to configure:

  • Email: Your Atlassian account email
  • API Token: Generated from Atlassian account settings
  • Base URL: Your Confluence instance URL (e.g., https://yourcompany.atlassian.net)

    • Configure these in the extension's options/settings page.

    Workflow

  • Navigate to any Confluence page
  • Click the extension icon to open the side panel
  • View the page JSON in the editor
  • Use find/replace to locate and modify content
  • Format JSON if needed for better readability
  • Save changes back to Confluence

    • Safety Features

  • Version conflict detection and automatic increment
  • JSON validation before save
  • Error handling with user-friendly messages
  • Non-destructive operations (requires explicit save)

Overview

Enhanced version of a Confluence visualization component that creates donut charts from lozenge data. Provides visual representation of status indicators and categories, making it easy to see at a glance the distribution of different statuses across your Blueprint pages.

Features

  • Donut Charts: Beautiful, interactive donut chart visualizations
  • Lozenge Tracking: Automatically processes lozenge (status badge) data
  • Visual Analytics: Clear visual breakdown of status distributions
  • Color-Coded: Uses lozenge colors for consistent representation
  • Interactive: Hover over segments for detailed information
  • Responsive: Adapts to different screen sizes

Use Cases

  • Visualize Blueprint standard distributions
  • Show compliance status at a glance
  • Track progress toward standards adoption
  • Compare different teams or time periods
  • Create executive dashboards

What's New in v2

  • Improved performance and rendering
  • Better color scheme handling
  • Enhanced interactivity
  • Mobile-responsive design
  • Integration with Blueprint Tracker data

Requirements

  • Atlassian Forge CLI
  • Node.js (v14 or higher)
  • Confluence Cloud instance
  • Admin access to install Forge apps

See Set up Forge for instructions.

Step 2: Deploy

# Clone the repository
git clone https://github.com/yourusername/lozenge-donut-chart-v2-claude-code.git
cd lozenge-donut-chart-v2-claude-code

Usage

Adding to a Confluence Page

  1. Edit any Confluence page
  2. Type / to open the macro menu
  3. Search for "Lozenge Donut Chart"
  4. Select the macro
  5. Configure your data source
  6. Publish the page

Configuration

When configuring the macro, you can specify:

  • Data Source: Connect to Google Sheets or use page lozenges
  • Filter Options: Show specific lozenge types or colors
  • Display Settings: Chart size, legend position, labels
  • Color Scheme: Use default lozenge colors or custom palette

Integration with Other Tools

Blueprint Tracker Integration

This chart component works seamlessly with the Blueprint Standard Adherence Master Tracker:

  1. Configure the donut chart to read from the tracker's Google Sheet
  2. Select the team tab you want to visualize
  3. The chart will automatically update when the tracker runs

Blueprint Standards Chart

Can be used alongside the Blueprint Standards Chart to provide multiple visualization perspectives on the same data.

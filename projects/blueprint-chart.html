Overview

Custom UI React component for Confluence that displays Blueprint standard adherence data with configurable chart visualizations. Integrates directly into Confluence pages as a macro, providing teams with visual insights into their Blueprint compliance.

Features

  • Confluence Macro: Embeds directly into Confluence pages
  • Custom UI: Built with React for interactive visualizations
  • Configurable: Easy-to-use configuration panel for customization
  • Real-time Data: Pulls latest data from Google Sheets
  • Multiple Chart Types: Bar charts, line graphs, and more
  • Responsive Design: Works on desktop and mobile

Use Cases

  • Display team Blueprint adherence metrics
  • Track progress over time
  • Compare standards across teams
  • Visualize compliance data in team spaces

Requirements

  • Atlassian Forge CLI
  • Node.js (v14 or higher)
  • Confluence Cloud instance
  • Admin access to install Forge apps

Installation

Step 1: Clone and Install

git clone https://github.com/yourusername/blueprint-standards-chart-custom-ui-react.git
cd blueprint-standards-chart-custom-ui-react
npm install

Step 2: Build the Frontend

npm run build

Step 3: Deploy

# Register the app (first time only)
forge register

# Deploy to Forge
forge deploy

# Install to your Confluence site
forge install

Usage

Adding to a Confluence Page

  1. Edit any Confluence page
  2. Type / to open the macro menu
  3. Search for "Blueprint Standards Chart"
  4. Select the macro
  5. Configure your data source and chart settings
  6. Publish the page

Configuration Options

  • Data Source: Connect to your Google Sheets data
  • Chart Type: Choose from bar, line, or area charts
  • Display Options: Customize colors, labels, and legends
  • Filters: Show data for specific teams or time periods

Local Development

# Run in development mode with live reload
forge tunnel

# View logs
forge logs

Making Changes

  1. Modify your app frontend by editing files in src/frontend/
  2. Modify your app backend by editing src/resolvers/index.js
  3. Build your changes: npm run build
  4. Deploy updates: forge deploy

Customization

Custom Chart Types

Add new chart types by editing src/frontend/components/ and importing additional chart libraries.

Styling

Customize the appearance by modifying CSS in the frontend components to match your organization's branding.

Data Sources

Extend the app to pull data from additional sources beyond Google Sheets (Jira, Confluence API, etc.).

Integration with Blueprint Tracker

This chart component works seamlessly with the Blueprint Standard Adherence Master Tracker. Configure it to read from the same Google Sheet that the tracker updates daily for automatic, up-to-date visualizations.

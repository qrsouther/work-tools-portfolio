Overview

Atlassian Forge app that provides a JSON editor interface within Confluence. Allows for structured data editing with configuration options directly in Confluence macros, enabling teams to manage structured content without leaving their documentation.

Features

  • JSON Editing: Intuitive interface for editing JSON data
  • Confluence Integration: Seamlessly embedded as a Confluence macro
  • Custom Macro: Easy to add to any Confluence page
  • Syntax Validation: Real-time validation of JSON syntax
  • Configuration Options: Customizable display and editing options
  • Version Control: Leverages Confluence's page versioning

Use Cases

  • Store and edit configuration data in Confluence
  • Maintain structured data alongside documentation
  • Create data-driven content in Confluence pages
  • Manage API configurations or settings
  • Collaborate on JSON-based content

Requirements

  • Atlassian Forge CLI
  • Node.js (v14 or higher)
  • Confluence Cloud instance
  • Admin access to install Forge apps

Usage

Adding the JSON Editor to a Page

  1. Edit any Confluence page
  2. Type / to open the macro menu
  3. Search for "JSON Editor"
  4. Select the macro
  5. Configure with your initial JSON data (optional)
  6. Publish the page

Editing JSON Data

  1. Click on the JSON Editor macro in view mode
  2. Edit the JSON in the editor interface
  3. The editor will validate syntax as you type
  4. Save changes to update the macro content

Configuration

When adding the macro, you can configure:

  • Initial Data: Starting JSON content
  • Read-Only Mode: Display JSON without editing
  • Display Options: Collapsed/expanded, theme, font size

Security Notes

  • JSON data is stored within Confluence pages
  • Respects Confluence page permissions
  • Only users with edit access can modify JSON data
  • Leverages Forge's security model

Troubleshooting

Syntax Errors

The editor will highlight syntax errors in real-time. Ensure your JSON is properly formatted before saving.

Cannot Save Changes

Verify you have edit permissions on the Confluence page.

Macro Not Appearing

Ensure the app is properly installed using forge install and check that your Confluence instance supports Forge apps.
